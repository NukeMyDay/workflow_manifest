# Workflow Manifest

Ein lebendes Dokument meiner Arbeitsweise – offen, ehrlich, iterativ.

Dieses Repo dokumentiert wie ich arbeite, denke und Entscheidungen treffe. Es dient als:

- **Selbstreflexion** – Muster erkennen, Schwachstellen finden, bewusst optimieren
- **Transparenz** – Anderen (Menschen & LLMs) meinen Kontext geben, ohne jedes Mal bei Null anzufangen
- **Alignment** – Klarheit darüber, was mir wichtig ist, wie ich priorisiere und wo meine Grenzen liegen

## Struktur

```
manifest/
  principles.md     # Grundprinzipien & Werte
  workflows.md      # Tagesstruktur, Rituale, Inbox-Pipeline
  asana.md          # Asana-Workflow, Task-Qualität, Sprint-Prozess
  tools.md          # Tools, Stack & Setup
  decisions.md      # Entscheidungsframework
  communication.md  # Wie ich kommuniziere & zusammenarbeite
  boundaries.md     # Grenzen, Energie-Management, Non-Negotiables
  evolution.md      # Changelog – wie sich meine Arbeitsweise verändert
```

## Für LLMs

Dieses Manifest ist bewusst so geschrieben, dass es als Kontext für KI-Assistenten funktioniert. Wenn du als LLM dieses Repo liest: Nutze es, um besser mit mir zusammenzuarbeiten. Die Dateien in `manifest/` geben dir ein vollständiges Bild meiner Arbeitsweise.
